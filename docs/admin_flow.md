# 🛡️ Flujo de Trabajo: Administrador

Guía para la gestión, moderación y control de calidad de la plataforma MicroCohorts.

## 1. Moderación de Cursos
- **Cola de Aprobación**: El administrador recibe notificaciones de cursos listos para ser publicados.
- **Acciones de Aprobación**:
    - **Aprobar**: Cambia el estado del curso a `APPROVED`. En este punto, el curso y sus grupos se vuelven visibles para los estudiantes en el catálogo público.
    - **Rechazar**: Devuelve el curso al profesor. El sistema solicita un motivo de rechazo que el profesor recibirá como feedback para corregir el curso.

## 2. Gestión de Usuarios
- **Listado Global**: Acceso a la base de datos de todos los usuarios registrados (Estudiantes y Profesores).
- **Control de Estado**: Capacidad para activar o desactivar usuarios (vía Toggle Status) por motivos de seguridad o incumplimiento de términos.
- **Auditoría**: Monitoreo de quién creó qué curso y quién aprobó cada módulo.

## 3. Seguridad y Configuración
- **Perfil de Super-Admin**: Gestión de datos personales y credenciales de acceso administrativo.
- **Avatar de Gestión**: Sistema de iniciales dinámico para una interfaz de gestión limpia y profesional.

## 4. Métricas y Supervisión (Pestaña Dashboard)
- **Cursos Totales**: Contador global de oferta educativa.
- **Usuarios Activos**: Estadística de crecimiento de la plataforma.
- **Métricas Financieras**: Monitoreo del flujo de dinero a través de las inscripciones completadas.
