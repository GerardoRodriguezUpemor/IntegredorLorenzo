# 👨‍🏫 Flujo de Trabajo: Proveedor (Profesor)

Guía completa para instructores sobre la gestión de oferta académica y finanzas en MicroCohorts.

## 1. Gestión de Cursos
- **Modo Borrador (Draft)**: Permite crear la estructura del curso (nombre, descripción) sin que sea visible para los estudiantes.
- **Configuración de Horarios**: Debes proponer **3 opciones de fechas y horas**. Estas opciones son las que los estudiantes votarán al momento de inscribirse.
- **Solicitud de Aprobación**: Una vez configurado el curso y sus fechas, se envía a revisión. El Administrador verificará el contenido antes de publicarlo.

## 2. Monitoreo de Grupos
- **Dashboard de Profesor**: Vista rápida de todos tus cursos y el estado de cada grupo.
- **Seguimiento de Inscritos**: Monitoreo en tiempo real de cuántos alumnos se han sumado (N de 5).
- **Resultados de Votación**: Acceso a la pestaña de finanzas para ver qué horarios están prefiriendo los estudiantes antes de que el grupo se cierre.

## 3. Finanzas y Reportes (EP4)
- **Activación de Cobro**: Un grupo solo es "válido" para remuneración cuando alcanza el cupo máximo de **5 alumnos**.
- **Reporte de Cohorte (PDF)**: Una vez que el grupo está lleno, se habilita un botón para descargar un reporte financiero detallado que incluye:
    - Lista de estudiantes inscritos.
    - Fechas de pago.
    - Ingreso total proyectado para el profesor.
    - Referencias de transacción.

## 4. Perfil y Especialidad
- **Avatar de Iniciales**: Representación profesional automática basada en tu nombre.
- **Especialidad Académica**: Puedes actualizar tu área de expertiz desde tu perfil; este dato se sincroniza automáticamente con la información que el estudiante ve en el catálogo.
- **Contacto**: Gestión de teléfono para que la administración o los estudiantes puedan contactarte según sea necesario.
